This is the Internet C++ port of XPacman.

To build xpacman, you will need the Internet C++ development system installed.

To run it you will need the Internet C++ virtual machine (ICVM) installed.
You should have icvm installed as /bin/icvm.


To build type:

	make


To run xpacman:


type:

	./xpacman 

or:

	icvm xpacman
